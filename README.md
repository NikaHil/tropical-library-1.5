# tropical-library-1.5

Сегодня в проекте была работа по улучшению внешнего вида и адаптации под разные устройства. Главным изменением стало оптимизация файла style.css из него удалены все повторяющиеся медиа-запросы и дублирующиеся стили. Также увеличены стикеры в карточках, на компьютерах они теперь имеют размер 64 пикселя вместо прежних 48, на планшетах 48 пикселей, а на маленьких телефонах 40 пикселей, для этого были добавлены inline-стили и специальные медиа-запросы. Графики в разделе статистики тоже были адаптированы: на ПК они отображаются в две колонки нормального размера, а на телефонах сворачиваются в одну колонку и уменьшаются до 200-220 пикселей, чтобы не вылезать за границы экрана
